线性同余法

约 1001 字大约 3 分钟

2025-12-04

概念

线性同余法（Linear Congruential Method，简称 LCM）是一种常见的伪随机数生成算法。它通过一个线性同余递推公式生成一系列伪随机数。该方法广泛应用于计算机模拟、统计、数值分析等领域，尤其是在随机数生成器中。

  • 优点：实现简单，速度快，适合用于简单的随机数生成。

  • 缺点：如果选择的参数不合适，可能会导致随机数序列的周期性，甚至完全重复。

线性同余法的基本递推公式如下：

{X0=seedXn=(AXn−1+C) mod M(n>=1)\begin{cases} X_{0}=seed\\ Xn=(AX_{n-1}+C)\,mod\,M \qquad (n>=1) \end{cases} {X0​=seedXn=(AXn−1​+C)modM(n>=1)​

式中Xn是伪随机序列，seed是种子变量，M是模数，A是乘数，C是增量**。

A,C,M的选取可以多种多样，为了方便计算，一般采用M=2^k混合同余法。当M为2的幂时，可以用位运算实现模运算，提高计算效率。

M越大，均值越接近(MIN+MAX)/2（MIN与MAX视参数而定），方差越接近1/12。

模运算（mod）确保生成的数值在 0 到 m−1m-1m−1 之间，即该算法会产生一个周期性序列。

线性同余法的最大周期是M，但一般情况下会小于M。要使周期达到最大，应该满足以下条件：

  • C和M互质；(没有共同的质因子，确保在一次完整的循环中，所有取值都能够被覆盖到)

  • M的所有质因子的积能整除A-1；(确保在一次完整的循环中，所有取值都能够被生成，以避免出现周期缩短的情况。)

  • 若M是4的倍数，则A-1也是；(为了避免周期缩短)

  • A,C,seed都比M小；(确保生成的伪随机数在取模运算时不超过上限M)

  • A,C是正整数。(有本事你不用正整数)

示例

比如：设M=16,A=5,C=3,seed=2。(1)C,M互质，(2)M的所有质因子「也就是2」的积「也就是2」可以整除A-1「也就是4」，(3)(4)(5)过于显然，略。满足条件。

# 自定义简单的随机数生成器（基于线性同余法）
def custom_random(seed):
    # 使用线性同余法 (Linear Congruential Generator) 来生成伪随机数
    a = 1664525
    c = 1013904223
    m = 2**32
    seed = (a * seed + c) % m
    return seed / m, seed  #

# 洗牌算法
def shuffle(arr):
    # 初始种子（可以自定义）
    seed = 12345  # 你可以设置一个固定的种子值
    for i in range(len(arr)-1, 0, -1):
        # 获取一个伪随机数
        rand, seed = custom_random(seed)
        j = int(rand * (i + 1))  # 随机选择一个索引
        # 交换元素
        arr[i], arr[j] = arr[j], arr[i]

# 原始数组
arr = [1, 2, 3, 4, 5, 6, 7, 8, 9]

# 打乱数组
shuffle(arr)

print("打乱后的数组:", arr)

提示

什么是归一化操作？

  • 即将得到的随机数落到[0,1)区间内

  • seed/m这是将生成的大整数 seed 归一化为一个标准的伪随机数 rand，使得它位于 [0, 1) 之间

seed / m 为什么在 [0, 1) 范围内？

当我们进行归一化操作时，实际上是将通过线性同余法生成的随机整数 seed 映射到一个更适合应用的浮动数范围 [0, 1)。要理解为什么 seed / m 会落在 [0, 1)，我们需要注意以下几点：

  • seed 是一个整数，m 是模数

  • seed 是通过 X_n = (a * X_{n-1} + c) % m 生成的，意味着 seed 的值永远是 0 到 m-1 之间的整数。

  • m 是模数，通常是一个很大的数（例如 2^32，即 4294967296）。它决定了随机数生成器生成的整数的最大值。

因此，seed 的值始终是一个小于 m 的整数，且位于 [0, m-1] 范围内。
