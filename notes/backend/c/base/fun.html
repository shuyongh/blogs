函数

约 5794 字大约 19 分钟

2025-12-10

函数是一组一起执行一个任务的语句。每个 C 程序都至少有一个函数，即主函数 main() ，所有简单的程序都可以定义其他额外的函数。

您可以把代码划分到不同的函数中。如何划分代码到不同的函数中是由您来决定的，但在逻辑上，划分通常是根据每个函数执行一个特定的任务来进行的。

函数声明告诉编译器函数的名称、返回类型和参数。函数定义提供了函数的实际主体。

定义函数

C 语言中的函数定义的一般形式如下：

return_type function_name( parameter list )
{
   body of the function
}

在 C 语言中，函数由一个函数头和一个函数主体组成。下面列出一个函数的所有组成部分：

  • 返回类型：一个函数可以返回一个值。return_type 是函数返回的值的数据类型。有些函数执行所需的操作而不返回值，在这种情况下，return_type 是关键字 void。
  • 函数名称：这是函数的实际名称。函数名和参数列表一起构成了函数签名。
  • 参数：参数就像是占位符。当函数被调用时，您向参数传递一个值，这个值被称为实际参数。参数列表包括函数参数的类型、顺序、数量。参数是可选的，也就是说，函数可能不包含参数。
  • 函数主体：函数主体包含一组定义函数执行任务的语句。

函数声明

函数声明会告诉编译器函数名称及如何调用函数。函数的实际主体可以单独定义。

函数声明包括以下几个部分：

return_type function_name( parameter list );

针对上面定义的函数 max()，以下是函数声明：

int max(int num1, int num2);

在函数声明中，参数的名称并不重要，只有参数的类型是必需的，因此下面也是有效的声明：

int max(int, int);

当您在一个源文件中定义函数且在另一个文件中调用函数时，函数声明是必需的。在这种情况下，您应该在调用函数的文件顶部声明函数。

命令行参数

执行程序时，可以从命令行传值给 C 程序。这些值被称为命令行参数，它们对程序很重要，特别是当您想从外部控制程序，而不是在代码内对这些值进行硬编码时，就显得尤为重要了。

命令行参数通过 main 函数的参数传递给程序。main 函数的原型可以是如下两种形式之一：

int main(int argc, char *argv[]);

或者:

int main(int argc, char **argv);
  • argc (argument count): 表示命令行参数的数量，包括程序名本身。因此，argc 至少为 1。
  • argv (argument vector): 是一个指向字符串数组的指针，其中每个字符串是一个命令行参数。数组的第一个元素（即 argv[0]）通常是程序的名称。接下来的元素是传递给程序的命令行参数。
    • 多个命令行参数之间用空格分隔，但是如果参数本身带有空格，那么传递参数的时候应把参数放置在双引号 "" 或单引号 '' 内部
    • 命令行参数通常是字符串，如果需要将其转换为数值类型，可以使用标准库函数如 atoi 或 strtol。

函数参数

如果函数要使用参数，则必须声明接受参数值的变量。这些变量称为函数的形式参数。

形式参数就像函数内的其他局部变量，在进入函数时被创建，退出函数时被销毁。

传值调用

该方法把参数的实际值复制给函数的形式参数。在这种情况下，修改函数内的形式参数不会影响实际参数。

默认情况下，C 语言使用传值调用方法来传递参数。一般来说，这意味着函数内的代码不会改变用于调用函数的实际参数

引用调用

通过指针传递方式，形参为指向实参地址的指针，当对形参的指向操作时，就相当于对实参本身进行的操作。

传递指针可以让多个函数访问指针所引用的对象，而不用把对象声明为全局可访问。

可变参数

有时，您可能会碰到这样的情况，您希望函数带有可变数量的参数，而不是预定义数量的参数。

C 语言为这种情况提供了一个解决方案，它允许您定义一个函数，能根据具体的需求接受可变数量的参数。

声明方式为：

int func_name(int arg1, ...);

其中，省略号 ... 表示可变参数列表。

请注意，函数 func() 最后一个参数写成省略号，即三个点号（...），省略号之前的那个参数是 int，代表了要传递的可变参数的总数。为了使用这个功能，您需要使用 stdarg.h 头文件，该文件提供了实现可变参数功能的函数和宏。具体步骤如下：

  • 定义一个函数，最后一个参数为省略号，省略号前面可以设置自定义参数。
  • 在函数定义中创建一个 va_list 类型变量，该类型是在 stdarg.h 头文件中定义的。
  • 使用 int 参数和 va_start() 宏来初始化 va_list 变量为一个参数列表。宏 va_start() 是在 stdarg.h 头文件中定义的。
  • 使用 va_arg() 宏和 va_list 变量来访问参数列表中的每个项。
  • 使用宏 va_end() 来清理赋予 va_list 变量的内存。

常用的宏有：

  • **va_start(ap, last_arg)**：初始化可变参数列表。ap 是一个 va_list 类型的变量，last_arg 是最后一个固定参数的名称（也就是可变参数列表之前的参数）。该宏将 ap 指向可变参数列表中的第一个参数。
  • **va_arg(ap, type)**：获取可变参数列表中的下一个参数。ap 是一个 va_list 类型的变量，type 是下一个参数的类型。该宏返回类型为 type 的值，并将 ap 指向下一个参数。
  • **va_end(ap)**：结束可变参数列表的访问。ap 是一个 va_list 类型的变量。该宏将 ap 置为 NULL。

示例

#include <stdio.h>
#include <stdarg.h>
 
double average(int num,...)
{
 
    va_list valist;
    double sum = 0.0;
    int i;
 
    /* 为 num 个参数初始化 valist */
    va_start(valist, num);
 
    /* 访问所有赋给 valist 的参数 */
    for (i = 0; i < num; i++)
    {
       sum += va_arg(valist, int);
    }
    /* 清理为 valist 保留的内存 */
    va_end(valist);
 
    return sum/num;
}
 
int main()
{
   printf("Average of 2, 3, 4, 5 = %f\n", average(4, 2,3,4,5));
   printf("Average of 5, 10, 15 = %f\n", average(3, 5,10,15));
}

函数修饰符

static

static 函数是文件作用域的函数，只能在定义它的源文件内部访问。

// 限制函数只在当前文件可见
static void internal_func(void) {
    // 只能在本文件内调用
    // 不会污染全局命名空间
}
  • 内部辅助函数
  • 实现细节隐藏
  • 避免命名冲突

inline

inline 是一个函数修饰符，建议编译器将函数代码内联展开，而不是进行普通的函数调用。

在现代编译器中，即使没有 inline 关键字，编译器也会自动内联它认为合适的函数。inline 更多是给编译器的明确提示。

普通函数调用：

int add(int a, int b) {
    return a + b;
}

int main() {
    int result = add(3, 5);  // 1. 压栈参数 2. 跳转 3. 执行 4. 返回
    return 0;
}

inline函数：

inline int add(int a, int b) {
    return a + b;
}

int main() {
    int result = add(3, 5);  // 可能被展开为：int result = 3 + 5;
    return 0;
}

优势：

  • 减少函数调用开销
  • 便于编译器优化
  • 适合小型频繁调用的函数

局限性

  • 只是建议，不是命令
  • 可能增加代码体积

函数指针

函数指针是指向函数的指针变量。

通常我们说的指针变量是指向一个整型、字符型或数组等变量，而函数指针是指向函数。

函数指针可以像一般函数一样，用于调用函数、传递参数。

函数指针类型的声明：

typedef int (*fun_ptr)(int,int); // 声明一个指向同样参数、返回值的函数指针类型

以下实例声明了函数指针变量 p，指向函数 max：

#include <stdio.h>
 
int max(int x, int y)
{
    return x > y ? x : y;
}
 
int main(void)
{
    /* p 是函数指针 */
    int (* p)(int, int) = & max; // &可以省略
    int a, b, c, d;
 
    printf("请输入三个数字:");
    scanf("%d %d %d", & a, & b, & c);
 
    /* 与直接调用函数等价，d = max(max(a, b), c) */
    d = p(p(a, b), c); 
 
    printf("最大的数字是: %d\n", d);
 
    return 0;
}

回调函数

函数指针变量可以作为某个函数的参数来使用的，回调函数就是一个通过函数指针调用的函数。

简单讲：回调函数是由别人的函数执行时调用你实现的函数。

基本回调

函数指针参数

适用场景:排序、遍历、转换

#include <stdlib.h>  
#include <stdio.h>
 // 回调函数类型定义
typedef void (*Callback)(int);

// 接受回调的函数
void process_numbers(int arr[], int size, Callback callback) {
    for (int i = 0; i < size; i++) {
        callback(arr[i]);  // 调用回调函数
    }
}
 // 回调函数1：打印数字
void print_number(int num) {
    printf("Number: %d\n", num);
}

// 回调函数2：打印平方
void print_square(int num) {
    printf("%d^2 = %d\n", num, num * num);
}

// 回调函数3：检查偶数
void check_even(int num) {
    if (num % 2 == 0) {
        printf("%d is even\n", num);
    }
}
int main() {
    int numbers[] = {1, 2, 3, 4, 5};
    int size = sizeof(numbers) / sizeof(numbers[0]);
    
    printf("=== 打印所有数字 ===\n");
    process_numbers(numbers, size, print_number);
    
    printf("\n=== 打印平方 ===\n");
    process_numbers(numbers, size, print_square);
    
    printf("\n=== 检查偶数 ===\n");
    process_numbers(numbers, size, check_even);
    
    return 0;
}

带上下文回调

上下文回调是指回调函数能够访问调用时的环境信息（上下文）。在C语言中，这通常通过传递一个void*指针实现，该指针可以指向任何类型的数据结构。

适用场景:对象方法、状态保持

#include <stdio.h>
#include <string.h>

// 带上下文数据的回调
typedef void (*CallbackWithContext)(const char*, void*);

// 处理字符串
void process_strings(const char* strings[], int count, 
                     CallbackWithContext callback, void* context) {
    for (int i = 0; i < count; i++) {
        callback(strings[i], context);
    }
}

// 回调：统计字符串长度
void count_length(const char* str, void* context) {
    int* total = (int*)context;
    *total += strlen(str);
    printf("String: %s (length: %zu)\n", str, strlen(str));
}

// 回调：查找包含子串的字符串
void find_substring(const char* str, void* context) {
    const char* substring = (const char*)context;
    if (strstr(str, substring) != NULL) {
        printf("Found '%s' in: %s\n", substring, str);
    }
}

// 回调：转换为大写
void to_uppercase(const char* str, void* context) {
    char buffer[100];
    strcpy(buffer, str);
    
    for (int i = 0; buffer[i]; i++) {
        if (buffer[i] >= 'a' && buffer[i] <= 'z') {
            buffer[i] -= 32;
        }
    }
    
    printf("Uppercase: %s\n", buffer);
}

int main() {
    const char* strings[] = {"hello", "world", "callback", "function", "example"};
    int count = sizeof(strings) / sizeof(strings[0]);
    
    printf("=== 统计总长度 ===\n");
    int total_length = 0;
    process_strings(strings, count, count_length, &total_length);
    printf("Total length: %d\n", total_length);
    
    printf("\n=== 查找包含 'll' 的字符串 ===\n");
    process_strings(strings, count, find_substring, "ll");
    
    printf("\n=== 转换为大写 ===\n");
    process_strings(strings, count, to_uppercase, NULL);
    
    return 0;
}

事件回调

响应特定事件

适用场景：GUI、游戏、消息系统

#include <stdio.h>
#include <stdlib.h>
#include <string.h>

// 事件类型
typedef enum {
    EVENT_CLICK,
    EVENT_KEYPRESS,
    EVENT_MOUSE_MOVE,
    EVENT_TIMER,
    EVENT_CUSTOM
} EventType;

// 事件结构
typedef struct {
    EventType type;
    void* data;
    int timestamp;
} Event;

// 事件回调函数类型
typedef void (*EventHandler)(Event*);

// 事件处理器结构
typedef struct {
    EventType type;
    EventHandler handler;
    void* user_data;
} EventHandlerEntry;

// 事件系统
typedef struct {
    EventHandlerEntry* handlers;
    int capacity;
    int count;
} EventSystem;

// 创建事件系统
EventSystem* create_event_system(int capacity) {
    EventSystem* system = malloc(sizeof(EventSystem));
    system->handlers = malloc(sizeof(EventHandlerEntry) * capacity);
    system->capacity = capacity;
    system->count = 0;
    return system;
}

// 注册事件处理器
void register_handler(EventSystem* system, EventType type, 
                      EventHandler handler, void* user_data) {
    if (system->count < system->capacity) {
        system->handlers[system->count].type = type;
        system->handlers[system->count].handler = handler;
        system->handlers[system->count].user_data = user_data;
        system->count++;
    }
}

// 触发事件
void trigger_event(EventSystem* system, Event* event) {
    for (int i = 0; i < system->count; i++) {
        if (system->handlers[i].type == event->type) {
            // 传递用户数据给处理器
            event->data = system->handlers[i].user_data;
            system->handlers[i].handler(event);
        }
    }
}

// 示例事件处理器
void handle_click(Event* event) {
    printf("[%d] Click event handled", event->timestamp);
    if (event->data) {
        printf(" with data: %s\n", (char*)event->data);
    } else {
        printf("\n");
    }
}

void handle_keypress(Event* event) {
    char* key = (char*)event->data;
    printf("[%d] Key pressed: %s\n", event->timestamp, key);
}

void handle_timer(Event* event) {
    int* counter = (int*)event->data;
    (*counter)++;
    printf("[%d] Timer tick: %d\n", event->timestamp, *counter);
}

int main() {
    // 创建事件系统
    EventSystem* system = create_event_system(10);
    
    // 注册事件处理器
    register_handler(system, EVENT_CLICK, handle_click, "Button1");
    register_handler(system, EVENT_KEYPRESS, handle_keypress, NULL);
    
    int timer_counter = 0;
    register_handler(system, EVENT_TIMER, handle_timer, &timer_counter);
    
    // 创建并触发事件
    Event click_event = {EVENT_CLICK, NULL, 1000};
    trigger_event(system, &click_event);
    
    Event key_event = {EVENT_KEYPRESS, "Enter", 1005};
    trigger_event(system, &key_event);
    
    Event timer_event = {EVENT_TIMER, NULL, 1010};
    for (int i = 0; i < 3; i++) {
        timer_event.timestamp = 1010 + i * 10;
        trigger_event(system, &timer_event);
    }
    
    // 清理
    free(system->handlers);
    free(system);
    
    return 0;
}

事件回调--GUI回调

#include <stdio.h>
#include <stdlib.h>
#include <string.h>

// 按钮回调函数类型
typedef void (*ButtonCallback)(const char* button_id, void* user_data);

// 按钮结构
typedef struct {
    char id[50];
    char label[50];
    int x, y;
    int width, height;
    ButtonCallback callback;
    void* user_data;
} Button;

// 按钮管理器
typedef struct {
    Button* buttons;
    int capacity;
    int count;
} ButtonManager;

// 创建按钮管理器
ButtonManager* create_button_manager(int capacity) {
    ButtonManager* manager = malloc(sizeof(ButtonManager));
    manager->buttons = malloc(sizeof(Button) * capacity);
    manager->capacity = capacity;
    manager->count = 0;
    return manager;
}

// 添加按钮
void add_button(ButtonManager* manager, const char* id, const char* label,
                int x, int y, int width, int height,
                ButtonCallback callback, void* user_data) {
    if (manager->count < manager->capacity) {
        Button* btn = &manager->buttons[manager->count];
        strcpy(btn->id, id);
        strcpy(btn->label, label);
        btn->x = x;
        btn->y = y;
        btn->width = width;
        btn->height = height;
        btn->callback = callback;
        btn->user_data = user_data;
        manager->count++;
    }
}

// 模拟点击按钮
void click_button(ButtonManager* manager, int click_x, int click_y) {
    for (int i = 0; i < manager->count; i++) {
        Button* btn = &manager->buttons[i];
        if (click_x >= btn->x && click_x <= btn->x + btn->width &&
            click_y >= btn->y && click_y <= btn->y + btn->height) {
            printf("Button '%s' clicked at (%d, %d)\n", 
                   btn->label, click_x, click_y);
            if (btn->callback) {
                btn->callback(btn->id, btn->user_data);
            }
            return;
        }
    }
    printf("No button clicked at (%d, %d)\n", click_x, click_y);
}

// 显示所有按钮
void display_buttons(ButtonManager* manager) {
    printf("\n=== Buttons ===\n");
    for (int i = 0; i < manager->count; i++) {
        Button* btn = &manager->buttons[i];
        printf("[%s] %s at (%d, %d) size %dx%d\n",
               btn->id, btn->label, btn->x, btn->y, 
               btn->width, btn->height);
    }
    printf("================\n");
}

// 回调函数示例
void save_callback(const char* button_id, void* user_data) {
    const char* filename = (const char*)user_data;
    printf("Saving to file: %s\n", filename);
}

void load_callback(const char* button_id, void* user_data) {
    int* counter = (int*)user_data;
    (*counter)++;
    printf("Load button clicked %d times\n", *counter);
}

void exit_callback(const char* button_id, void* user_data) {
    printf("Exit button clicked. Goodbye!\n");
    exit(0);
}

void print_callback(const char* button_id, void* user_data) {
    printf("Printing document...\n");
}

int main() {
    // 创建按钮管理器
    ButtonManager* manager = create_button_manager(10);
    
    // 添加按钮
    add_button(manager, "btn1", "Save", 10, 10, 80, 30, 
               save_callback, "data.txt");
    
    int load_counter = 0;
    add_button(manager, "btn2", "Load", 100, 10, 80, 30, 
               load_callback, &load_counter);
    
    add_button(manager, "btn3", "Print", 190, 10, 80, 30, 
               print_callback, NULL);
    
    add_button(manager, "btn4", "Exit", 280, 10, 80, 30, 
               exit_callback, NULL);
    
    // 显示按钮
    display_buttons(manager);
    
    // 模拟点击
    printf("\n=== Simulating Clicks ===\n");
    click_button(manager, 50, 25);   // 点击Save按钮
    click_button(manager, 140, 25);  // 点击Load按钮
    click_button(manager, 140, 25);  // 再次点击Load按钮
    click_button(manager, 230, 25);  // 点击Print按钮
    click_button(manager, 500, 500); // 点击空白处
    
    // 清理
    free(manager->buttons);
    free(manager);
    
    return 0;
}

异步回调

延迟执行

适用场景:I/O操作、定时器、网络

#include <stdio.h>
#include <stdlib.h>
#include <string.h>
#include <unistd.h>
#include <pthread.h>

// 任务回调类型
typedef void (*TaskCallback)(void* result, void* user_data);
typedef void* (*TaskFunction)(void* arg);

// 异步任务结构
typedef struct {
    TaskFunction function;
    void* arg;
    TaskCallback callback;
    void* user_data;
    void* result;
    int completed;
} AsyncTask;

// 任务队列
typedef struct {
    AsyncTask* tasks;
    int capacity;
    int count;
    pthread_mutex_t mutex;
    pthread_cond_t cond;
    pthread_t* threads;
    int thread_count;
    int running;
} TaskQueue;

// 创建任务队列
TaskQueue* create_task_queue(int capacity, int thread_count) {
    TaskQueue* queue = malloc(sizeof(TaskQueue));
    queue->tasks = malloc(sizeof(AsyncTask) * capacity);
    queue->capacity = capacity;
    queue->count = 0;
    queue->thread_count = thread_count;
    queue->running =0;
        queue->threads = malloc(sizeof(pthread_t) * thread_count);
    
    pthread_mutex_init(&queue->mutex, NULL);
    pthread_cond_init(&queue->cond, NULL);
    
    return queue;
}

// 工作线程函数
void* worker_thread(void* arg) {
    TaskQueue* queue = (TaskQueue*)arg;
    
    while (queue->running) {
        pthread_mutex_lock(&queue->mutex);
        
        // 等待任务
        while (queue->count == 0 && queue->running) {
            pthread_cond_wait(&queue->cond, &queue->mutex);
        }
        
        if (!queue->running) {
            pthread_mutex_unlock(&queue->mutex);
            break;
        }
        
        // 获取任务
        AsyncTask task = queue->tasks[0];
        
        // 移动队列
        for (int i = 0; i < queue->count - 1; i++) {
            queue->tasks[i] = queue->tasks[i + 1];
        }
        queue->count--;
        
        pthread_mutex_unlock(&queue->mutex);
        
        // 执行任务
        void* result = task.function(task.arg);
        
        // 调用回调
        if (task.callback) {
            task.callback(result, task.user_data);
        }
        
        // 清理
        free(task.arg);
    }
    
    return NULL;
}

// 启动工作线程
void start_workers(TaskQueue* queue) {
    queue->running = 1;
    for (int i = 0; i < queue->thread_count; i++) {
        pthread_create(&queue->threads[i], NULL, worker_thread, queue);
    }
}

// 停止工作线程
void stop_workers(TaskQueue* queue) {
    queue->running = 0;
    pthread_cond_broadcast(&queue->cond);
    
    for (int i = 0; i < queue->thread_count; i++) {
        pthread_join(queue->threads[i], NULL);
    }
}

// 提交任务
void submit_task(TaskQueue* queue, TaskFunction func, void* arg,
                 TaskCallback callback, void* user_data) {
    pthread_mutex_lock(&queue->mutex);
    
    if (queue->count < queue->capacity) {
        AsyncTask task = {
            .function = func,
            .arg = arg,
            .callback = callback,
            .user_data = user_data,
            .result = NULL,
            .completed = 0
        };
        
        queue->tasks[queue->count++] = task;
        pthread_cond_signal(&queue->cond);
    }
    
    pthread_mutex_unlock(&queue->mutex);
}

    printf("Calculating %dth prime...\n", n);
    
    // 简单的质数计算
    int count = 0;
    int num = 2;
    
    while (count < n) {
        int is_prime = 1;
        for (int i = 2; i * i <= num; i++) {
            if (num % i == 0) {
                is_prime = 0;
                break;
            }
        }
        if (is_prime) {
            count++;
        }
        num++;
    }
    
    int* result = malloc(sizeof(int));
    *result = num - 1;
    
    return result;
}

#include <stdio.h>
#include <stdlib.h>
#include <string.h>
#include <time.h>
#include <unistd.h>
#include <pthread.h>

// 定时器回调类型
typedef void (*TimerCallback)(void* user_data);

// 定时器结构
typedef struct Timer {
    int id;
    int interval_ms;      // 间隔时间（毫秒）
    int repeat;           // 是否重复
    TimerCallback callback;
    void* user_data;
    long long next_fire;  // 下次触发时间
    struct Timer* next;
} Timer;

// 定时器管理器
typedef struct {
    Timer* head;
    pthread_mutex_t mutex;
    pthread_t thread;
    int running;
    int next_id;
} TimerManager;

// 创建定时器管理器
TimerManager* create_timer_manager() {
    TimerManager* manager = malloc(sizeof(TimerManager));
    manager->head = NULL;
    manager->running = 1;
    manager->next_id = 1;
    pthread_mutex_init(&manager->mutex, NULL);
    return manager;
}

// 获取当前时间（毫秒）
long long current_time_ms() {
    struct timespec ts;
    clock_gettime(CLOCK_MONOTONIC, &ts);
    return ts.tv_sec * 1000LL + ts.tv_nsec / 1000000LL;
}

// 添加定时器
int add_timer(TimerManager* manager, int interval_ms, int repeat,
              TimerCallback callback, void* user_data) {
    pthread_mutex_lock(&manager->mutex);
    
    Timer* timer = malloc(sizeof(Timer));
    timer->id = manager->next_id++;
    timer->interval_ms = interval_ms;
    timer->repeat = repeat;
    timer->callback = callback;
    timer->user_data = user_data;
    timer->next_fire = current_time_ms() + interval_ms;
    timer->next = manager->head;
    
    manager->head = timer;
    
    pthread_mutex_unlock(&manager->mutex);
    
    return timer->id;
}

// 移除定时器
void remove_timer(TimerManager* manager, int timer_id) {
    pthread_mutex_lock(&manager->mutex);
    
    Timer** ptr = &manager->head;
    while (*ptr) {
        if ((*ptr)->id == timer_id) {
            Timer* to_free = *ptr;
            *ptr = (*ptr)->next;
            free(to_free);
            break;
        }
        ptr = &(*ptr)->next;
    }
    
    pthread_mutex_unlock(&manager->mutex);
}

// 定时器线程函数
void* timer_thread(void* arg) {
    TimerManager* manager = (TimerManager*)arg;
    
    while (manager->running) {
        pthread_mutex_lock(&manager->mutex);
        
        long long now = current_time_ms();
        Timer* current = manager->head;
        Timer* prev = NULL;
        
        while (current) {
            if (now >= current->next_fire) {
                // 触发定时器
                if (current->callback) {
                    current->callback(current->user_data);
                }
                
                if (current->repeat) {
                    // 更新下次触发时间
                    current->next_fire = now + current->interval_ms;
                    prev = current;
                    current = current->next;
                } else {
                    // 一次性定时器，移除
                    Timer* to_free = current;
                    if (prev) {
                        prev->next = current->next;
                        current = current->next;
                    } else {
                        manager->head = current->next;
                        current = manager->head;
                    }
                    free(to_free);
                }
            } else {
                prev = current;
                current = current->next;
            }
        }
        
        pthread_mutex_unlock(&manager->mutex);
        
        // 休眠10毫秒
        usleep(10000);
    }
    
    return NULL;
}

// 启动定时器管理器
void start_timer_manager(TimerManager* manager) {
    pthread_create(&manager->thread, NULL, timer_thread, manager);
}

// 停止定时器管理器
void stop_timer_manager(TimerManager* manager) {
    manager->running = 0;
    pthread_join(manager->thread, NULL);
    
    // 清理所有定时器
    pthread_mutex_lock(&manager->mutex);
    Timer* current = manager->head;
    while (current) {
        Timer* next = current->next;
        free(current);
        current = next;
    }
    manager->head = NULL;
    pthread_mutex_unlock(&manager->mutex);
    
    pthread_mutex_destroy(&manager->mutex);
    free(manager);
}

typedef struct {
    const char* name;
    const char* version;
    void (*init)(void);
    void (*process)(const char* data, void (*callback)(const char*));
    void (*cleanup)(void);
} Plugin;

// 插件管理器
typedef struct {
    Plugin** plugins;
    int capacity;
    int count;
} PluginManager;

// 创建插件管理器
PluginManager* create_plugin_manager(int capacity) {
    PluginManager* manager = malloc(sizeof(PluginManager));
    manager->plugins = malloc(sizeof(Plugin*) * capacity);
    manager->capacity = capacity;
    manager->count = 0;
    return manager;
}

// 加载插件
int load_plugin(PluginManager* manager, const char* path) {
    void* handle = dlopen(path, RTLD_LAZY);
    if (!handle) {
        fprintf(stderr, "Cannot load plugin: %s\n", dlerror());
        return 0;
    }
    
    // 获取插件符号
    Plugin* (*get_plugin)(void) = dlsym(handle, "get_plugin");
    if (!get_plugin) {
        fprintf(stderr, "Not a valid plugin: %s\n", path);
        dlclose(handle);
        return 0;
    }
    
    Plugin* plugin = get_plugin();
    if (manager->count < manager->capacity) {
        manager->plugins[manager->count++] = plugin;
        printf("Loaded plugin: %s v%s\n", plugin->name, plugin->version);
        return 1;
    }
    
    return 0;
}

// 初始化所有插件
void init_plugins(PluginManager* manager) {
    for (int i = 0; i < manager->count; i++) {
        if (manager->plugins[i]->init) {
            manager->plugins[i]->init();
        }
    }
}

// 处理数据
void process_data(PluginManager* manager, const char* data,
                  void (*result_callback)(const char*)) {
    for (int i = 0; i < manager->count; i++) {
        if (manager->plugins[i]->process) {
            printf("\nProcessing with plugin: %s\n", manager->plugins[i]->name);
            manager->plugins[i]->process(data, result_callback);
        }
    }
}

// 清理插件
void cleanup_plugins(PluginManager* manager) {
    for (int i = 0; i < manager->count; i++) {
        if (manager->plugins[i]->cleanup) {
            manager->plugins[i]->cleanup();
        }
    }
}

    printf("  Result: %s\n", result);
