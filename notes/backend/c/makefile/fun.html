外部调用

shell

调用当前系统的shell命令

$(shell command)

参数：

  • command：要执行的 shell 命令

返回值：

  • 返回命令的标准输出（stdout）
  • 自动去除尾部的换行符

提示

  • 在 Windows 上，命令通过 cmd.exe 执行
  • 在 Unix-like 系统上，命令通过 /bin/sh 执行

流程控制

foreach

将 list 中的每个元素依次代入 var，对每个元素展开 text，将所有展开结果用空格连接成一个字符串

$(foreach var, list, text)

## 示例
# 注意：结果中的空格来自 text 的展开
$(foreach x, a b, $(x))    # 返回: "a b"（两个单词）
$(foreach x, a b, $(x) )   # 返回: "a b "（注意结尾空格）

参数

  • var - 循环变量
  • list - 遍历列表
  • text - 展开文本

返回值

  • 所有展开文本连接成一个字符串

目录操作

wildcard

查找当前目录下匹配的文件

$(wildcard pattern...)

参数：

  • pattern：一个或多个通配符模式
  • 支持的通配符：
    • *：匹配任意数量的任意字符
    • ?：匹配单个任意字符
    • [...]：字符集匹配（如 [abc]）
    • [^...] 或 [!...]：排除字符集

返回值：

  • 返回匹配模式的文件名列表（空格分隔）
  • 如果没有匹配的文件，返回空字符串

dir

获取文件目录

$(dir names...)

参数：

  • names：一个或多个文件名或路径

返回值：

  • 返回每个参数目录部分的列表
  • 如果参数没有目录部分（如 filename），返回 ./

文本操作

patsubst

在 text 中查找匹配 pattern 的部分，替换为 replacement

$(patsubst pattern,replacement,text)
## $(patsubst %.c,%.o,main.c utils.c config.c)

# subst: 直接文本替换
$(subst 查找文本,替换文本,字符串)

参数：

  • pattern：包含通配符 % 的模式字符串
  • replacement：替换字符串，可以包含 % 引用匹配的部分
  • text：要处理的文本（空格分隔的单词列表）

返回值：

  • 返回将 text 中匹配 pattern 的部分替换为 replacement 后的新字符串
